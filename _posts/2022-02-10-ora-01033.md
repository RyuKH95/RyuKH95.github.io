cmd 켜서
```terminal
C:\>SQLPLUS
```

접속 계정으로 로그인 안될 시
```terminal
/as sysdba

recover database;  --복구 완료 이후
alter database open;
```

if ORA-00600 에러 발생 시 :
  ```terminal
  recover database using backup controlfile until cancel;  --복구 전으로 취소
  cancel
  alter database open resetlogs; --REDO LOG 리셋
  shutdown immediate;
  startup
  ```
else :
  ```terminal
  shutdown immediate;
  startup
  ```
