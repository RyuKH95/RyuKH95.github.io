#### [문제](https://www.acmicpc.net/problem/10171){:target="_blank" .btn .btn--primary}
아래 예제와 같이 고양이를 출력하시오.  
```terminal
\    /\  
 )  ( ')  
(  /  )  
 \(__)|  
```

#### 코드
```java
import java.io.*; 
class Main {
    public static void main(String[] args) {
        System.out.println("\\    /\\");
        System.out.println(" )  ( \')");
        System.out.println("(  /  )");
        System.out.println(" \\(__)|");
    }
}
```
