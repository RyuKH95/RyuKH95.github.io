#### [문제](https://www.acmicpc.net/problem/10172){:target="_blank" .btn .btn--primary}
아래 예제와 같이 개를 출력하시오.  
```terminal
|\_/|  
|q p|   /}  
( 0 )"""\  
|"^"`    |  
||_/=\\__|  
```

#### 코드
```java
import java.io.*; 
class Main {
    public static void main(String[] args) {
        System.out.println("|\\_/|");
        System.out.println("|q p|   /}");
        System.out.println("( 0 )\"\"\"\\");
        System.out.println("|\"^\"`    |");
        System.out.println("||_/=\\\\__|");
    }
}
```
